Source File Format

A source file is a spreadsheet containing the following columns:

oid

Required.The OID of the player. For information about where to find the OID, see tools.

name

Required1.The player's real name. This is currently not used, but is useful to recognize OIDs in the spreadsheets.

nickname

Required1.The player's in game nickname. This is shown everywhere this extension adds data.

level

Required1.The player's level. This is shown everywhere this extension adds data.

…

Optional.You can create other columns at will. These will be optional, and will be added to the extratags of the source, but only for the player in the row where the data is written. If the column name is also added in extratags, the data in extratags is overwritten.

Example: the spreadsheet has an 'anomaly' column. The data in this column will overwrite any 'anomaly' values in extratags for the individual players.

Note 1: These values are required once. That is, it's perfectly fine for this extension if you don't include this data the second time a player is used. See the compatibility page for more information.
