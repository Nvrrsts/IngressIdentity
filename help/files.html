Files

This extension makes use of manifest files and source files. This page explains what those are exactly.

Source files

A source file contains a list of players. This list usually looks like this:

oid name nickname level
999999999999999999999 Dummy Dummyson dummy 0
103568659333550762891 Roland Jarvis __JARVIS__ 16
108200572857719809031 Oliver Lynton-Wolfe Oliver ?

This table links the Google+ profile (the oid column) with the in-game intel (nickname and level). The name column isn't used. It is there to help you identify a row in the table. A more detailed explanation of the source file format can be found here.

Manifest files

A manifest file is a list of source files. It adds information to a source, for example to which faction the players in the source file belong. The manifest also defines how often the extension checks for updates. A typical manifest looks like this:

lastupdated tag faction key refresh
2014010101 Dummy unknown 9999999999999999999999999999999999 0
2014051601 13M Resistance resistance &gid=2 100
2014051601 13M Enlightened enlightened &gid=5 100
2014042903 REC Resistance resistance &gid=6 100
2014043001 REC Enlightened enlightened &gid=7 100

A more detailed explanation of manifest files can be found here.
