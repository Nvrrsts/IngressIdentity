Tools

This page contains some tools that can help you when using this extension.

Manifest key

Simply enter the manifest URL below, and this page will give you the key to use.

OID

An OID is a unique identifier. People, pages, events and communities each have an OID. Finding it for events and communities is easy, but it can be tricky for people.

Events and Communities

Events and communities have a simple URL structure:

https://plus.google.com/events/OID/...

https://plus.google.com/communities/OID/...

Note that there might be a /u/0 in the URL.

People

People are slightly more tricky, as they can choose a custom URL. You'll have to find a link to the person's profile, instead of taking the URL of the profile page itself. To see what we mean, take a look at Klue's profile. If you look at the URL, you will see the following:

https://plus.google.com/u/0/+KlueS/posts

There's no OID there. If you hover over any of the links to Klue's profile, however, you will see a link with an OID:

https://plus.google.com/u/0/110350977702120778591

Tool

This tool filters out the OID for any person, community or event. Note that it won't work with the personalised URLs like Klue's above, you'll have to get the link.
