Install IngressIdentity

This page contains information on how to install the IngressIdentity browser extension.

Google Chrome

Looks like you're not using Google Chrome. You can of course still download the Chrome extension, but don't try to install it anywhere else.

On Google Chrome (or other browsers with access to the Chrome Webstore) you can easily install the extension by clicking here or by going to the webstore page here.
