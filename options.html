IngressIdentity Options

This options page allows you to configure the manifests to use when identifying people on G+. It also provides some tools to help acquiring data to put in manifest files or source files.

Success! The data was successfully reloaded.
Error! Something went wrong when reloading the data.
Success! The manifest file was successfully added.
Warning! The given manifest is already loaded.
Error! Something went wrong when adding the manifest.
Success! The manifest file was successfully removed.
Warning! The given manifest was not loaded.
Error! Something went wrong when removing the manifest.
