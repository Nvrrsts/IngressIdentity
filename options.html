Settings

This section contains some settings for the extension. Some settings will lead the extension to request extra permissions.

Automatically push source updates to the webpages

An extra permission is required in order to send messages from the extension to the tabs where its content scripts run. Without this permission, the extension can only reply to messages sent from those scripts.

If this option is disabled (default), the G+ page will update when the first request has been sent after that update became availble.

Enabling this extension makes that operation happen automatically, without the need for the content script to send a request first.

The needed extra permission is 'Access to your tabs and browsing activity'. This extension does not require it by default because some people don't trust extensions that require this permission.

You only see the prompt to grant the permission once. Chrome remembers your choice if you granted the permission and will automatically grant it again the next time this extension requests it.

Disabling this option revokes the requested permission. You can check this by clicking the 'Permissions' link next to this extension on chrome://extensions.
