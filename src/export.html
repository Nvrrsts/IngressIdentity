IngressIdentity Data Exporter

This page tries to fetch Ingress agent data from a Google+ community. You can modify it before exporting it to a Google Drive Spreadsheet.

Options

The following buttons allow you to enable some features for the exporter:

Try to extract agent name from the Google+ name if unknown.

Filter out existing entries for this community.

Add a header and dummy row to the table.

Result

Simply select the text, and copy paste it into a Google Drive Spreadsheet
